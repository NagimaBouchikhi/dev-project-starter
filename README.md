# DevProject Starter (Nom Provisoire)

**STATUT : EN DÉVELOPPEMENT ACTIF (WIP)**

Ce projet est actuellement en construction. L'objectif est de créer un outil pédagogique pour aider les développeurs à structurer leurs projets “sans improvisation”.

## Objectif du projet

DevProject Starter est une **application web visuelle** destinée aux développeurs (solo ou en équipe) pour :

- Choisir une **architecture adaptée** à leur projet
- Visualiser **la structure des dossiers** et comprendre le rôle de chaque partie
- Déterminer **l’ordre des étapes** de développement
- Découper le travail **clairement en équipe**
- (Phase future) Générer des **prompts IA précis et contextualisés**

Ce projet est pensé pour améliorer la **compréhension et l’organisation**, et non remplacer la réflexion humaine.

## Fonctionnalités prévues

### MVP (en cours)

- Formulaire de description du projet (type, techno, solo/équipe)
- Recommandation d’architecture visuelle (MVVM, Clean, etc.)
- Arbre de dossiers interactif avec explications
- Timeline des étapes de développement
- Découpage des tâches pour les équipes

### Bonus (à venir)
- Génération de prompts IA contextualisés
- Export PDF / Markdown des structures et prompts
- Tableau de suivi équipe

##  Tech Stack

- **Frontend :** React  
- **UI :** Tailwind CSS / composants React  
- **Gestion de l’état :** hooks et logique métier simple  
- **Données :** JSON ou fichiers statiques (pas de backend pour le MVP)
